# Insurance Salary Prediction ML Project

This project uses machine learning to predict salary/income patterns based on insurance-related customer data through a complete regression pipeline.

## Project Overview

The objective of this project is to analyze insurance data, extract useful insights, and build a machine learning model to predict salary/income using regression techniques.

## Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker
* Region
* Insurance Charges
* Salary (Target)

## Machine Learning Pipeline

### 1. Exploratory Data Analysis (EDA)

* Data inspection
* Data visualization
* Feature distribution analysis
* Correlation analysis

### 2. Data Cleaning

* Duplicate removal
* Missing value checking
* Data consistency validation

### 3. Data Preprocessing

* Label encoding
* One-hot encoding
* Feature scaling

### 4. Feature Engineering

* Feature transformation
* Feature selection using statistical methods

### 5. Model Training

* Train-test split
* Linear Regression implementation

### 6. Model Evaluation

* R² Score
* Adjusted R²
* Performance analysis
